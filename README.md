<h2><a class="anchor" id="overview"></a>Overview</h2>

The Online Courses Landscape Analysis project focuses on analyzing an online courses dataset sourced from Kaggle to understand trends in course offerings, learner engagement, and skill demand. Using Power BI, the project transforms raw educational data into an interactive and insightful dashboard that highlights key performance indicators such as total courses, total instructors, total views, and average course duration.
The report provides a structured walkthrough of category-wise and sub-category-wise course distribution, in-demand skills, language preferences, course duration impact on viewership, and instructor performance. Through intuitive visuals and business-focused insights, the project demonstrates how data analytics can support strategic decisions for ed-tech platforms, instructors, and content planners by identifying growth opportunities and improving learner engagement.

<h2><a class="anchor" id="visual-analysis-breakdown"></a>Visual Analysis Breakdown</h2>

**1. Category Slicer**
Allows users to filter the entire report by major course categories such as Computer Science, Business, Data Science, and others for focused analysis.

**2. Course Type Popularity**
Displays the distribution of courses by type (Course, Specialization, Professional Certificate), highlighting which format is most commonly offered.

**3. Courses per Sub-Category and Language**
Shows the number of courses across different sub-categories segmented by language, helping identify dominant topics and language preferences.

**4. Demanding Skills (Word Cloud)**
Visualizes the most frequently taught skills, emphasizing high-demand skills like Python, Machine Learning, and Data Analysis.

**5. Viewership Based on Course Length**
Illustrates how average viewership varies with course duration, helping understand learner engagement trends.

**6. Top 5 Categories by Average Views**
Ranks course categories based on average number of views, identifying the most engaging subject areas.

**7. Average Viewers by Subtitle Language Count**
Analyzes how the availability of subtitles in multiple languages impacts average viewership.

**8. Courses by Language**
Represents the proportion of courses offered in different languages, highlighting language dominance on the platform.

**9. Top 3 Instructors**
Compares instructors based on the number of courses offered, showcasing leading contributors.

**10. Relationship Between Number of Skills and Course Duration**
Examines how the number of skills covered in a course relates to its duration across different categories.

**11. Sub-Category Selector**
Enables dynamic filtering of visuals by specific sub-categories such as Algorithms, Animal Health, and Basic Science.
